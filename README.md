# 🕸️ SpiderWeb: Get to Know Spiders from Different Regions

## 📌 Project Overview
SpiderWeb is a web-based platform designed to provide informative and visually engaging content about spiders from different regions around the world. The website showcases spider characteristics, habitats, and danger levels, while maintaining an interactive and thematic user experience.

---

## 🎯 Objectives
- Provide educational content about spiders across global regions
- Present information in a structured, visually engaging format
- Apply core web development concepts (HTML, CSS, JavaScript)
- Demonstrate responsive design and interactive UI components

---

## 🧠 Concept & Theme
The website follows a **“spider-cave” aesthetic**, incorporating:
- Web-like UI elements
- Dark-themed backgrounds
- Animated transitions and effects
- Consistent footer across all pages

---

## 🏗️ Website Structure

### 1. 🏠 Home Page (`home.html`)
- Hero section with slideshow of spiders
- Welcome message
- Overview of the website purpose

---

### 2. 🌍 Regions Page (`region.html`)
- Dropdown navigation menu for regions
- Overview of spider habitats by region
- Environmental descriptions
- Slideshow of region-specific spiders
- Tabular data showing spider frequency

---

### 3. 🕷️ Spider Page (`spider.html`)
- Grid layout displaying different spiders
- Clickable items with detailed pop-up information
- Images and videos embedded
- Common household spider references

---

### 4. 📩 Contact Page (`contact.html`)
- Contact form including:
  - Name
  - Email
  - Message
- Enables user interaction and feedback

---

## 🛠️ Technologies Used

### 🔹 Frontend
- HTML5 (Semantic structure)

- JavaScript (Interactivity)

### 🔹 Key Concepts Applied
- Semantic tags (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`)

- Animations and transitions
- Dropdown navigation menu
- Slideshow functionality
- Form handling (basic validation)
